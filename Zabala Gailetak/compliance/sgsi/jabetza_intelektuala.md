# Jabetza Intelektualaren Politika

**Helburua:** Zabala Gailetak-en eta hirugarrenen jabetza intelektuala babestea.

## 1. Software Lizentziak
- Lizentziapeko softwarea bakarrik erabili daiteke.
- Debekatuta dago software pirata edo "crack"-eatua instalatzea.
- Aldian behin inbentarioa egingo da lizentziak egiaztatzeko.

## 2. Kode Iturria
- Enpresan garatutako kode guztia Zabala Gailetak-en jabetza da.
- Ez da kodea biltegi publikoetara igo behar (GitHub Public) baimenik gabe.

## 3. Hirugarrenen Eskubideak
- Ez erabili copyright bidez babestutako irudi, testu edo materialik baimenik gabe.
- Open Source liburutegiak erabiltzean, lizentziaren baldintzak (MIT, GPL, Apache) errespetatu.

## 4. Sekretu Industriala
- Errezetak eta prozesu industrialak sekretu gisa tratatuko dira (KONFIDENTZIALA maila).
