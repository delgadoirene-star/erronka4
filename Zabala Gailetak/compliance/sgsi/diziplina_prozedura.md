# Diziplina Prozedura (Segurtasun Urratsak)

**Helburua:** Segurtasun politiken urraketen aurrean neurriak hartzea.

## 1. Ikerketa
- Segurtasun intzidente bat gertatzean, CISO-k eta HR-k ikerketa egingo dute.
- Ebidentziak bilduko dira (logs, emails, bideoak) GDPR errespetatuz.
- Langileari entzuteko aukera emango zaio.

## 2. Larritasun Mailak eta Zehapenak
- **Arina:** (Adib: PC desblokeatuta uztea behin) -> Ahozko ohartarazpena.
- **Larria:** (Adib: Pasahitza partekatzea, software ez baimendua) -> Idatzizko zigorra / Soldata gabe eta lan gabe.
- **Oso Larria:** (Adib: Datuen lapurreta, sabotajea) -> Kaleratzea eta lege-ekintzak.

## 3. Erregistroa
- Hartutako neurriak langilearen espedientean gordeko dira.
